# 🛡️ Key Risk Indicator (KRI) Dashboard

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)]https://kridashboardproject-ffkhqtudhwal896u7wncyi.streamlit.app/

A dynamic web application built with **Streamlit**, **Plotly**, and **TextBlob** to monitor operational risks, visualize threshold breaches, and perform natural language sentiment analysis on risk logs.

## 🚀 Live Demo
Access the live dashboard here: https://kridashboardproject-ffkhqtudhwal896u7wncyi.streamlit.app/

## 🌟 Features
* **Automated Data Processing:** Instantly reads and formats risk metric logs.
* **NLP Sentiment Analysis:** Uses TextBlob to evaluate the tone of operational logs (ranges from -1.0 highly critical to +1.0 stable).
* **Dynamic Threshold Mapping:** Categorizes risks into Safe (Green), Warning (Amber), and Critical (Red) based on predefined logic.
* **Interactive Visualizations:** Built-in Plotly charts for tracking risk values over time and mapping sentiment polarity.

## 🛠️ Tech Stack
* **Python** (Backend & Logic)
* **Streamlit** (Web Framework)
* **Plotly** (Interactive Data Visualization)
* **TextBlob** (Natural Language Processing / Sentiment)
* **Pandas** (Data Manipulation)
